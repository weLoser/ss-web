Linux使用教程

(任何一步遇到问题，可进群讨论.QQ群:414301879)

一、安装shadowsocks软件

  1. 在Ubuntu/Debian下：

    2. sudo apt-get install python-pip

    sudo pip install shadowsocks

  2. 在CentOs/RedHat下：

    3. sudo yum install python-pip

    sudo yum install shadowsocks

二、创建配置文件

查看帐号信息,并创建配置文件：config.json
(此处直接新建一个名为config.json的文本即可,文本内填入以下内容，并把红色内容改为你自己帐号对应的信息)

{
"server": "my_server_ip",
"server_port": 0000,
"local_address":"127.0.0.1",
"local_port":1080,
"password": "my_passwd",
"timeout":600,
"method":"aes-256-cfb"
}

三、启动shadowsocks

运行shadowsocks使用命令:
sslocal -c config.json
(若不在config.json的目录下执行此命令，记得在config.json前加上路径)
若不想看到输出信息，可使用命令:
nohup sslocal -c config.json >/dev/null 2>&1 &

  2. 然后将SwitchySharp.crx拖到扩展程序页面,点击添加：


  3. 加入成功后右上角会显示，右击-选项，打开配置选项卡。

  4. 选择新建情景模式，将链接复制https://raw.githubusercontent.com/KyonLi/ss-pac/master/ss.pac到PAC网址中并保存。
